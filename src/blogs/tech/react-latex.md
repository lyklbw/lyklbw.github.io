# React中支持LaTeX

你可以在Markdown中直接书写LaTeX公式，例如：

## 行内公式
这是一个行内公式：$E = mc^2$，爱因斯坦的质能方程。

## 块级公式
块级公式会单独成行显示：

$$E = mc^2$$

## 复杂公式示例

### 二次方程
$$ax^2 + bx + c = 0$$

### 积分公式
$$\int_{-\infty}^{\infty} e^{-x^2} dx = \sqrt{\pi}$$

### 矩阵
$$
\begin{bmatrix}
a & b \\
c & d
\end{bmatrix}
$$

### 求和公式
$$\sum_{i=1}^{n} x_i = x_1 + x_2 + \cdots + x_n$$

### 分数
$$\frac{a}{b} + \frac{c}{d} = \frac{ad + bc}{bd}$$
